# 🖼️CodexArt

## Description
**CodexArt** est une application web développée dans le cadre d’un projet collaboratif,suivant l'architecture React d'un projet Monorepo JS <br>
enseignée à la **Wild Code School**. 

Inspirée par notre passion commune pour l’art.
CodexArt a pour objectif de rendre l’art plus accessible, en associant technologie et culture au service d’une expérience enrichissante pour l’utilisateur.
Codex Art met en avant l’art sous toutes ses formes.<br>
Elle permet de découvrir les œuvres depuis une galerie virtuelle, de rechercher des musées et d’explorer les lieux culturels disponibles autour de soi.

## 🛠️ Installation

- Clone le dépôt :<br>
 ```
  git clone git@github.com:wildcodeschool-2025-03/JS-RemoteFR-Vert-ASquad-codex-art-P2.git
 ```
   
- Installer les dépendances :<br>
**cd {nom_de_fichier_clone}** <br>
**npm install**<br>

- Démarrez le client et le serveur :
<br>**npm run dev**
